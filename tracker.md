Project Tracker is a local-first kanban board with MCP integration for AI-assisted development. Phase 1 (shared-surface context foundation) is in progress — shipping projectPrompt and renderStatus to eliminate private-memory rot. The north star: if the human can't see it and correct it in the surface where they'd naturally encounter it, the agent shouldn't trust it. Current work is tracked on the Development Board.
